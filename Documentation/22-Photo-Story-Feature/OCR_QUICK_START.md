# OCR Feature - Quick Start Guide

## üöÄ Getting Started with OCR

### What is OCR?
OCR (Optical Character Recognition) extracts text from images. Use it to:
- Read text from photos
- Convert book pages to digital text
- Extract text from signs, documents, or screenshots
- Use extracted text to inspire AI-generated stories

## üì∏ How to Use OCR

### Step 1: Open Photo Story Modal
Click **"Create from Photo"** on the home page

### Step 2: Select OCR Mode
Toggle to **"Text Extraction"** mode (right button)

### Step 3: Add Your Photo
- **Take Photo**: Use camera to capture text
- **Upload Photo**: Select existing photo

### Step 4: Extract Text
Click **"Extract Text from Image"** button

### Step 5: Use the Results
Three options:
1. **Use as Story Context** - Generate AI story with extracted text
2. **Copy Text** - Copy to clipboard for other uses
3. **Re-extract** - Try again with different photo

## üí° Tips for Best Results

### ‚úÖ DO:
- Use good lighting
- Keep camera steady
- Hold perpendicular to text
- Use high contrast (dark text on light background)
- Ensure text is in focus

### ‚ùå DON'T:
- Use blurry images
- Photograph at extreme angles
- Use low light conditions
- Include too much background clutter

## üêõ Troubleshooting

**No text detected?**
- Ensure image has clear, readable text
- Try better lighting or focus

**Low accuracy?**
- Retake photo with camera perpendicular to text
- Ensure good contrast

**Slow extraction?**
- First use takes longer (initializing)
- Subsequent uses are faster

## üéì Pro Tips

1. **For Books**: Flatten pages, use good lighting
2. **For Signs**: Stand directly in front, avoid angles
3. **For Handwriting**: Use clear, block letters
4. **For Screens**: Increase brightness, avoid glare
5. **For Documents**: Use scanner mode if available
