================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "keytool not found"
Solution: Install Java JDK and add to PATH
          Download from: https://www.oracle.com/java/technologies/downloads/

Problem: "Keystore password incorrect"
Solution: Check android/keystore.properties file
          Re-run setup-keystore.bat if needed

Problem: Icons not showing
Solution: Clean build and try again:
          cd android
          gradlew clean
          cd ..
          build-beta-apk.bat
