# Authentication Page

This page provides login and signup functionality for the Imaginary Worlds app with a magical storybook theme.

## Features

- Tab-based interface for switching between Login and Sign Up
- Form validation for required fields
- Account type selection (Child, Parent, Educator)
- Responsive design that works on all device sizes
- Magical animations and glassmorphism effects
- Accessibility features including proper focus states

## Components

- `AuthPage.tsx` - Main authentication page component
- `AuthPage.css` - Additional styling for the authentication page

## Routes

- `/auth` - Authentication page
- `/` - Redirects to authentication page (default route)

## Styling

The authentication page uses the existing magical storybook design system with:
- Glassmorphism cards and inputs
- Gradient backgrounds
- Floating animations
- Magical pulse effects
- Responsive layouts
